# 📜 Invitation from Wish

6명의 왕자가 당신에게 청혼했습니다.  
레트로 터미널 스타일의 인터랙티브 웹 스토리텔링 경험.

---

## ⚙ Tech Stack

- **Frontend**: Vanilla JavaScript (ES6), HTML5 Canvas, CSS3
- **Audio**: Tone.js (v14.8.49)
- **Server**: Node.js
- **Image Processing**: Python 3 + Pillow

---

## 🎮 About

CRT 모니터 스타일의 터미널 인터페이스로 6명의 왕자들의 이야기를 탐험하는 웹 애플리케이션입니다.

**주요 기능:**
- 1-bit 픽셀 아트 이미지 처리
- A Minor Sarabande 배경음악
- 키보드/터치 인터랙션
- iOS 모바일 최적화
- NCT WISH 이스터에그

---

## 🚀 Quick Start

```bash
# Install Python dependencies
pip install -r requirements.txt

# (Optional) Process images
python process_images.py

# Start server
node server.js

# Open browser
http://localhost:8080
```
